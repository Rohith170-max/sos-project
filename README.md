# SOS Emergency Alert System

A real-time emergency alert web application that allows users to send SOS alerts with live location tracking. Built using HTML, CSS, JavaScript, Node.js, Express, and Socket.IO.
# How to Run

## Prerequisites

* Node.js (v18 or later)
* npm (comes with Node.js)

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/sos-emergency-alert-system.git
```

2. Open the project folder:

```bash
cd sos-emergency-alert-system
```

3. Navigate to the server folder:

```bash
cd server
```

4. Install dependencies:

```bash
npm install
```

5. Start the server:

```bash
node server.js
```

If the project uses Nodemon:

```bash
npm run dev
```

6. Open the client:

* Open `client/index.html` in your browser, **or**
* Use the Live Server extension in VS Code.

7. The application will be available at:

```
http://localhost:3000
```

## Features

* Send SOS emergency alerts
* Real-time communication using Socket.IO
* Live location sharing
* Responsive user interface
* Emergency notification dashboard
